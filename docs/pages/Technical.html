Technical information

For this project we used several important things to develop the game.

ECS

Entity-Component-System is a design pattern that allows you to create a game engine with a good performance and a good scalability.

It's based on three main concepts:

  • Entities: are the objects in the game, they are just an ID.
  • Components: are the data of the entities, they are just data.
  • Systems: are the logic of the game, they are the functions that manipulate the data of the components.

Raylib

Raylib is a simple and easy-to-use library to learn videogame programming. It's written in plain C language and it's based on OpenGL.

It has been developed to be easy to learn and use, so it's a good choice for a little video game like this one.
