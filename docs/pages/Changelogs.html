Air-Type - Changelogs

Release S3: v0.3

Menus and update coordinator

Third release on the Air-Type project. Adds a lot of graphics/client part to the project and game. The next step is to finish server part and add the game logic to the server.

Features:

  • Added Pause Menu
  • Added Settings Menu
  • Handle change of keyboard input in Controls panel of settings
  • Added Video panel in settings
  • Added changement of resolution, display FPS and change FPS max in Video
  • Added option doc with the build to generate and open a doxygen documentation website

Fixes:

  • Lots of Doxygen comments in code
  • Cleaning & adding functions in coordinator
  • Now the Game class uses the settings component in the coordinator
  • CMake now compile shared libraries instead of static ones

Release S2: v0.2

ECS and documentation

Second release of the Air-Type project. Adds ECS (Entity-Component-System) to the project.

Features:

  • Added ECS (Entity-Component-System) to the project (v.1)
  • Created this documentation github pages
  • Added Doxygen to the project
  • Started the graphic part of the game with Core and Game classes

Fixes:

  • Changed the structure & compilation architecture of the project

Release S1: v0.1

Architecture and CI/CD

First release of the Air-Type project. Adds the architecture of the project and the CI/CD. We are going to use the ECS architectural pattern for the game. See you next release !

Features:

  • Added the CI/CD:
    • Checks everytime there is a commit if it builds on Windows and Linux
    • Handles the release of version in main
    • Push to the Epitech repository when everything is good in main
  • Added the architecture of the project
  • Created all of the CMake for compiling everything

Others:

  • Research on the ECS architectural pattern
